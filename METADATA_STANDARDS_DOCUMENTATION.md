# Metadata Standards Documentation

## Candidate ID

Unique identifier for each candidate.

Example:
C001

## Job ID

Unique identifier for each job opening.

Example:
J001

## Question ID

Unique identifier for each screening question.

Example:
Q001

## Timestamp

Stores the exact date and time of the interaction.

Format:
YYYY-MM-DD HH:MM:SS

## Confidence Level

Speech-to-text confidence score.

Range:
0.0 – 1.0
