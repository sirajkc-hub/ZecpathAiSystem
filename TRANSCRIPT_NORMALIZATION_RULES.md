# Transcript Normalization Rules

## Lowercase Conversion

Convert all text to lowercase.

Example:
Python Developer
→
python developer

## Remove Extra Spaces

Normalize spacing.

## Remove Special Symbols

Remove unnecessary punctuation and symbols.

## Standardize Dates

Convert dates to a single format.

Example:
20/06/2026
→
2026-06-20

## Expand Common Abbreviations

yrs → years
exp → experience
