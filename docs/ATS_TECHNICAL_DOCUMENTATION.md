# ATS Technical Documentation

## System Overview

The ATS system processes resumes and compares them with job descriptions.

## Modules

1. Resume Parser
2. JD Parser
3. Section Classifier
4. Skill Extractor
5. Experience Parser
6. Education Parser
7. Semantic Matcher
8. ATS Scorer
9. Candidate Ranker
10. Fairness Engine

## Outputs

* ATS Score
* Candidate Ranking
* Shortlisting Recommendation
