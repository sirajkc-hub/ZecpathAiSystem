# ATS Scoring Logic

## Inputs

* Skill Score
* Experience Score
* Education Score
* Semantic Similarity

## Weight Distribution

Data Scientist:

Skill Match = 40%

Experience = 25%

Education = 15%

Semantic Similarity = 20%

## Formula

Final ATS Score =
(Skill × Weight) +
(Experience × Weight) +
(Education × Weight) +
(Semantic × Weight)
